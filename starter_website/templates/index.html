Welcome to the Character Manager!

  • This site intends to make your Dungeons & Dragons games more organized and streamlined by providing a way for dungeon masters and players to keep track of each character's guild, class, spells, and stats. It provides an easy way for all players to reference details of each character in the game. The Character Manager makes a great utility for the dungeon master as they already have so much to keep track of. Adventure on!
